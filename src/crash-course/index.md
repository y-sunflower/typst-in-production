!!! info

    The official Typst documentation website has [its own tutorial](https://typst.app/docs/tutorial/), which is very good too and worth reading!

<br>

This crash course is doable by **anyone**, even if you have never used Typst before. It starts from the ground up and gradually introduces more complex concepts. It's currently organized into 3 parts:

- [Foundations](./lesson-1.md)
- [Scripting](./lesson-2.md)
- [Templating](./lesson-3.md)

All of them have **exercises** at the end to help you practice your new knowledge.

You're ready to start, but first quickly check [how to use Typst](./how-to-use-typst.md).
